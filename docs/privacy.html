Axon Companion – Privacy Policy

Last updated: 08-23-2025

Overview

Axon Companion adds local memory to ChatGPT by recording the text you write and the assistant’s replies on chatgpt.com and storing them locally on your device. The extension does not send your data to external servers operated by the developer or third parties.

Data We Access

  • Website content on https://chatgpt.com/* (your prompts and assistant responses) — used solely to build a local session log and a local YAML profile for personalization.

How Your Data Is Used

  • Data is processed locally by the extension and a local daemon (running on localhost over WebSocket).
  • No analytics, tracking, or advertising code.
  • No data is sold or transferred to third parties.

Storage & Retention

  • Session logs and the YAML profile are stored on your machine under the extension’s configured storage folder (e.g., ./storage/ in the Axon project).
  • You can delete these files at any time to erase your data.

Permissions Rationale

  • activeTab / tabs – to run only on the active ChatGPT tab and inject the content script after you interact.
  • webNavigation / webRequest – to detect when the ChatGPT page is loaded/changed so capturing works reliably. These events are not used to collect browsing history.
  • storage – to save local settings (e.g., enable/disable memory injection).
  • Host permission https://chatgpt.com/* – restricts operation strictly to ChatGPT.

Remote Code

The extension is shipped as a bundled package (no external scripts are loaded at runtime; no eval of remote strings). If you obtained Axon from the Chrome Web Store, all code executing in the extension is included in the published package.

User Control

  • You may disable the extension at any time from the browser’s extensions page.
  • Local files can be removed manually to clear your history/profile.
